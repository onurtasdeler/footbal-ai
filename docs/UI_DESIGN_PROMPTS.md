# UI Tasarım Promptları - Bahis Analiz Uygulaması

Her sayfa için AI görsel tasarım araçlarında (Midjourney, DALL-E, Figma AI, v0.dev) kullanılacak promptlar.

---

## 1. Dashboard / Ana Sayfa

### Türkçe Prompt
```
Modern bahis analiz dashboard tasarımı. Koyu tema (dark mode). Sol tarafta navigasyon menüsü. Üstte günün önemli maçları kartları. Ortada value bet fırsatları listesi yeşil vurgulu. Sağ tarafta canlı skor widget'ı. Alt kısımda performans grafikleri. Minimalist, profesyonel fintech tarzı. Mor ve yeşil aksan renkleri. Glassmorphism efektleri. 1920x1080 web dashboard.
```

### English Prompt
```
Modern betting analysis dashboard design. Dark theme with deep purple background. Left sidebar navigation with icons. Top section showing today's featured matches as cards with team logos, odds, and prediction percentages. Center area displaying value bet opportunities list with green highlight indicators. Right sidebar with live scores widget. Bottom section with performance analytics charts and ROI graphs. Minimalist, professional fintech style. Purple and green accent colors. Glassmorphism effects. Subtle gradients. Clean typography. 1920x1080 web dashboard UI.
```

## 2. Maç Detay Sayfası

### Türkçe Prompt
```
Futbol maç analiz detay sayfası. Üstte iki takım logosu karşı karşıya, ortada VS yazısı. Altında bahis oranları kartları (1X2, Over/Under, BTTS). Sol kolonda ev sahibi takım istatistikleri, sağ kolonda deplasman. Ortada kafa kafaya geçmiş maçlar tablosu. Alt kısımda tahmin yüzdeleri pasta grafik ve bar chart. Prediction skoru büyük ve vurgulu. Koyu tema, yeşil ve turuncu aksan. Modern spor uygulaması tarzı.
```

### English Prompt
```
Football match analysis detail page. Header section with two team logos facing each other, VS text in center with match date and league info. Below header: betting odds cards showing 1X2, Over/Under 2.5, BTTS options with bookmaker odds comparison. Left column: home team statistics (form, goals scored/conceded, clean sheets). Right column: away team statistics mirrored layout. Center section: head-to-head history table with last 5 meetings. Bottom area: prediction percentages as donut chart and probability bar charts. Large highlighted prediction score with confidence meter. Dark theme, green and orange accents. Modern sports app style. Data visualization focused. 1440x900 desktop UI.
```

## 3. Value Bet Listesi Sayfası

### Türkçe Prompt
```
Value bet fırsatları liste sayfası. Tablo formatında maçlar: takımlar, lig, bahis tipi, oran, tahmin yüzdesi, EV değeri. EV pozitif olanlar yeşil satır, yüksek value olanlar parlak yeşil badge. Filtreleme: lig, tarih, bahis tipi, minimum EV. Üstte özet kartları: toplam fırsat sayısı, ortalama EV, en iyi fırsat. Sıralama seçenekleri. Koyu tema, veri odaklı tasarım. Bloomberg terminal estetiği.
```

### English Prompt
```
Value bet opportunities list page. Table format showing matches: team names with small logos, league badge, bet type tag, bookmaker odds, prediction percentage, calculated EV value, confidence score. Positive EV rows highlighted in subtle green, high value bets (EV>10%) with bright green badge. Filter bar: league dropdown, date range picker, bet type selector, minimum EV slider. Top summary cards: total opportunities count, average EV percentage, best opportunity preview. Sorting options for each column. Dark theme with data-focused design. Bloomberg terminal aesthetic. Monospace numbers. Clean grid layout. 1920x1080 data dashboard.
```

## 4. Takım Analiz Sayfası

### Türkçe Prompt
```
Futbol takım profil ve analiz sayfası. Üstte takım logosu, ismi, lig pozisyonu banner. Sol tarafta sezon istatistikleri kartları: maç, galibiyet, gol, puan. Ortada form grafiği son 10 maç (W/D/L renkli). Sağda güçlü/zayıf yönler radar chart. Alt kısımda gol atılan dakikalar bar chart, ev/deplasman karşılaştırma. En altta son maçlar listesi. Takım renkleriyle aksan. Koyu tema, spor analitik tarzı.
```

### English Prompt
```
Football team profile and analysis page. Header banner with team logo large, team name, current league position badge, and season selector dropdown. Left section: season statistics cards showing matches played, wins/draws/losses, goals scored/conceded, points, goal difference. Center section: form chart visualizing last 10 matches as colored circles (green W, yellow D, red L) with opponent info on hover. Right section: strengths/weaknesses radar chart (attack, defense, possession, set pieces, discipline). Bottom left: goals scored by minute intervals bar chart. Bottom center: home vs away performance comparison. Bottom right: recent matches list with results. Team primary color as accent throughout. Dark theme base. Sports analytics style. 1440x900.
```

## 5. Kafa Kafaya (H2H) Sayfası

### Türkçe Prompt
```
İki takım kafa kafaya karşılaştırma sayfası. Üstte iki takım logosu, aralarında VS. İstatistik karşılaştırma: toplam maç, galibiyetler her iki taraf, beraberlik. Ortada son karşılaşmalar timeline, skorlar ve tarihler. Alt kısımda istatistik karşılaştırma bar chart: gol ortalaması, şut, korner, kart. Trend analizi: son 5 maçta kim daha iyi. Koyu tema, kırmızı-mavi iki taraf renklendirme.
```

### English Prompt
```
Head-to-head team comparison page. Top section: two team logos large with VS symbol between, team names below. Statistics comparison row: total meetings number, home team wins, draws, away team wins as segmented bar. Center section: historical meetings timeline showing last 10 matches chronologically with scores, dates, competition badges. Expandable rows for match details. Bottom left: statistical comparison horizontal bar chart comparing goals avg, shots, corners, cards, possession. Bottom right: recent trend analysis showing last 5 H2H performance. Two-tone color scheme: home team color (blue) vs away team color (red). Dark theme base. Split screen feel. Sports broadcast style. 1440x900.
```

## 6. Canlı Maçlar Sayfası

### Türkçe Prompt
```
Canlı maç takip sayfası. Grid layout maç kartları. Her kartta: iki takım logosu, canlı skor büyük font, dakika (yanıp sönen), son olay ikonu (gol/kart). Kartlarda mini istatistik: şut, top hakimiyeti yüzde bar. Üstte filtre: tüm ligler, sadece favoriler. Sağ tarafta seçili maç detaylı istatistik panel. Canlı güncellenen animasyonlar. Koyu tema, yeşil canlı indikatör, kırmızı gol animasyonu.
```

### English Prompt
```
Live matches tracking page. Grid layout with match cards. Each card contains: two team logos small, live score in large bold font, current minute with pulsing live indicator dot, last event icon (goal/card/substitution). Mini statistics bar in card: shots on target, possession percentage as thin progress bar. Top filter bar: all leagues, favorites only toggle, search. Right side panel: selected match detailed statistics (possession, shots, corners, cards live updating). Real-time update animations. Dark theme. Green live indicator accents. Red pulse animation on goals. Yellow flash on cards. Smooth transitions. Sports broadcast overlay style. 1920x1080.
```

## 7. Bahis Oranları Karşılaştırma Sayfası

### Türkçe Prompt
```
Bahis oranları karşılaştırma sayfası. Tablo: satırlarda bahisçiler (Bet365, 1xBet, vs), sütunlarda bahis tipleri (1, X, 2, O2.5, U2.5). En iyi oranlar yeşil vurgulu. Üstte maç bilgisi header. Oran değişim ok ikonları (yukarı yeşil, aşağı kırmızı). Oran geçmişi mini sparkline grafik. Tek tıkla en iyi oran seçimi. Koyu tema, finansal veri görselleştirme tarzı.
```

### English Prompt
```
Betting odds comparison page. Match info header with team logos, date, league. Main comparison table: rows for each bookmaker (Bet365, Betfair, 1xBet, Unibet, William Hill), columns for bet types (Home, Draw, Away, Over 2.5, Under 2.5, BTTS Yes/No). Best odds highlighted in green cell. Odds movement indicators: up arrow green, down arrow red. Historical odds mini sparkline chart on hover. One-click best odds selection with external link. Probability conversion toggle. Margin percentage display. Dark theme. Financial data visualization style. Clean table design with subtle grid lines. Monospace numbers for easy scanning. 1440x900.
```

## 8. Tahmin Performans / ROI Sayfası

### Türkçe Prompt
```
Bahis performans takip sayfası. Üstte özet kartları: toplam bahis, kazanma oranı %, toplam ROI %, kar/zarar. Ana grafik: zaman serisi çizgi grafik ROI trendi. Pasta grafik: bahis tipi dağılımı. Bar chart: lig bazında performans. Tablo: son bahisler listesi sonuçlarla. Filtreler: tarih aralığı, bahis tipi, lig. Yeşil kar, kırmızı zarar renklendirme. Koyu tema, finans dashboard tarzı.
```

### English Prompt
```
Betting performance tracking page. Top summary cards: total bets placed, win rate percentage with trend arrow, total ROI percentage, profit/loss amount in currency. Main chart: time series line graph showing ROI trend over selected period with profit/loss area fill. Secondary charts row: pie chart for bet type distribution, bar chart for performance by league. Bottom section: recent bets table with date, match, bet type, odds, stake, result (W/L), profit/loss. Filter bar: date range, bet type, league, minimum odds. Green for profit, red for loss color coding throughout. Dark theme. Finance dashboard style. Professional analytics aesthetic. 1920x1080.
```

Loading state skeleton screens collection. Dashboard skeleton: gray placeholder boxes for cards with animated shimmer effect. List skeleton: row placeholders with avatar circle and text lines. Match card skeleton: team logo circles, score rectangle, stats bars. Chart skeleton: axis lines with area placeholder. Left to right shine animation. Dark theme with dark gray (#2a2a2a) placeholders on darker background (#1a1a1a). Subtle pulse animation. Modern loading state design. Show multiple skeleton variations in one frame. 1920x1080.
